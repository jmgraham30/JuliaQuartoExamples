A Plot

Let’s plot \(f(x)=x^2\) over \([-1,1]\).

  • Plot
  • Code
x = -1.0:0.01:1.0;
y = x.^2;
plot(x,y,legend=false,xlab="x",ylab="y",lw=2,,title=L"Plot of $y=x^2$.")
